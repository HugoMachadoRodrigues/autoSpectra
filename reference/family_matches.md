# Test whether a family matches a sensor type

Test whether a family matches a sensor type

## Usage

``` r
family_matches(fam, sensor_type, moisture = "agnostic")
```

## Arguments

- fam:

  Family list from `model_registry`

- sensor_type:

  Character: `"visnir"` or `"mir"`

- moisture:

  Ignored; retained for backward compatibility

## Value

Logical
